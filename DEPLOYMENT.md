# GitHub Pages Deployment Guide

This repository contains a portfolio landing page in `index.html`.

## 1. Enable GitHub Pages

1. Open this repository on GitHub.
2. Go to **Settings**.
3. Click **Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
5. Save.

## 2. Portfolio URL

After GitHub Pages finishes deployment, the website should be available at:

```text
https://ikhwanmanab.github.io/ikhwanmanab/
```

## 3. Recommended Files to Upload

The portfolio is already prepared to use these paths:

```text
images/passportfoto.png
/doc/resume.pdf
```

Upload the files using the same folder and filename structure.
