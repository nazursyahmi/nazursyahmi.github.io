# Simple Git Workflow

This is a basic step-by-step flow from getting the latest changes to pushing your work.

## 1. Pull the latest changes

Before editing, make sure your local branch is up to date.

```bash
git pull
```

## 2. Check the current status

See which files are changed, added, or deleted.

```bash
git status
```

## 3. Add your changes

Add all changed files:

```bash
git add .
```

Or add one specific file:

```bash
git add filename.html
```

## 4. Commit your changes

Save your changes with a short message.

```bash
git commit -m "Update project files"
```

## 5. Push your changes

Upload your committed changes to GitHub.

```bash
git push
```

## Quick command summary

```bash
git pull
git status
git add .
git commit -m "Your commit message"
git push
```
