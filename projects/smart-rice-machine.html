A COVID-19 NGO project that automated rice aid distribution using RFID cards, beneficiary allocation data, and a connected dispensing machine.

COVID-19

NGO aid distribution use case

Overview

Built for controlled aid distribution.

The system made rice distribution more traceable and less dependent on manual checking at the dispensing point.

Problem

Rice aid distribution required manual recipient verification, allocation checking, and physical control during a period where efficient aid handling mattered.

  • Manual recipient verification
  • Manual allocation checking
  • Limited automation at dispensing point
  • Harder collection tracking

Solution

Each recipient received an RFID card. The machine reads the card, checks the server allocation, then dispenses the approved rice quantity.

  • Admin registers recipient profile
  • System assigns rice allocation
  • RFID links recipient to allocation
  • Machine dispenses based on instruction
System Flow

Identity, validation, and dispensing.

The machine does not dispense blindly. It checks recipient data and allocation first.

01 Admin

Admin registers recipient information and rice allocation.

02 RFID

Each card links a recipient to their profile and allocation.

03 Scan

The machine reads the card and sends the request to the server.

04 Validate

The backend checks eligibility and available collection quantity.

05 Dispense

The machine dispenses the approved rice amount.
