Project Overview

The system was created to make rice distribution more controlled, traceable, and automated for eligible Asnaf recipients.

🎯

Problem

Rice aid distribution required manual checking, manual allocation, and physical control. This created operational friction, especially during COVID-19.

  • Manual recipient verification
  • Manual rice allocation checking
  • Limited automation at dispensing point
  • Harder to track who already received aid
✅

Solution

Each Asnaf recipient received an RFID card. The machine reads the card, checks the server allocation, then dispenses the approved rice quantity.

  • Admin registers recipient profile
  • System assigns rice allocation in kilograms
  • RFID card links recipient to allocation
  • Machine dispenses rice based on server instruction

System Flow

End-to-End Architecture

The machine does not simply dispense rice blindly. It checks recipient data and allocation first.

Admin Registration → Recipient Profile → RFID Card → Machine Scan → Server Check → KG Allocation → Dispense Rice

User Journey

The process is simple for recipients but controlled at the backend.

Admin Registers Asnaf

Admin records recipient information and assigns the rice allocation in the system.

Recipient Receives RFID Card

Each recipient receives a unique RFID card linked to their profile and allocation.

Recipient Scans Card at Machine

The smart machine reads the RFID card and sends the request to the server.

Server Validates Allocation

The backend checks eligibility, allocation quantity, and whether the recipient can collect.

Machine Dispenses Rice

The server gives instruction to the machine, and the drawer opens after dispensing the allocated rice.

Business & Community Impact

The main value was not just technical. The system helped make aid distribution more reliable and controlled.

Controlled

Rice was dispensed based on registered allocation instead of uncontrolled manual distribution.

Traceable

Each recipient used an RFID card, making the collection process easier to validate and track.

Automated

The machine reduced manual handling by reading card data, checking allocation, and dispensing rice automatically.
